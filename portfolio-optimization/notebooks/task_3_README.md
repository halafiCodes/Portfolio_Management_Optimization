# Task 3 — Forecast Future Market Trends

## Objective
Generate 6–12 month TSLA forecasts using the best model from Task 2, visualize uncertainty, and translate results into insights.

## Notebook
- notebooks/task_3.ipynb

## Key Steps
- Load the saved Task 2 model
- Generate a 12‑month forecast with confidence intervals
- Plot historical data, test predictions, and future forecasts
- Write trend analysis, opportunities, risks, and reliability assessment

## Outputs
- Forecast plot with confidence intervals
- Trend analysis summary
- Opportunities and risks list

## How to Run
Open the notebook and run all cells from top to bottom.
